设置方法

  • 下载最新的把版本，解压到对应目录

    C:\CodeBlocks\

  • 环境变量 用户 PATH 添加

    C:\CodeBlocks\MinGW64\bin;

  • 运行 CMD 开启窗口，输入 gcc -v 就可以使用

  • 也可以在 CodeBlocks 设置对应的编译器

不添加path变量使用

  • 建立文件 C:\CodeBlocks\MinGW6\mingwvars.bat
